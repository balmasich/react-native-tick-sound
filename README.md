# react-native-tick-sound

Tick sound

## Installation

```sh
yarn add react-native-tick-sound
cd ios
pod install
```

## Usage

```js
import { playSound, stopSound } from 'react-native-tick-sound';
const rate = 1;
playSound(rate);

// ...

```
